#### return
return 语句用于终止函数执行，并指定返回给调用函数的值，只能在函数体内使用
```js
return [expression];
```
#### `continue`
```js
continue [label]; // 执行 label 标签所指循环的下一轮循环
```
#### `break`

```
break [label]; // 跳出标签 label 所在的循环/代码块，继续执行后面的代码
```
